προδύνω
produnō

Greek verb

Last edited: 12/2/2024, 4:22:31 PM
Primary meaning to set earlier
Literal translation to set
Variants
προδύνω produnō
Edited by
Thomas Peeters
to set earlier astronomical event

(...) καὶ ὅταν προδύνῃ τοῦ ἡλίου καὶ ὅταν ἐπικαταδύνῃ τῷ ἡλίῷ, ἐπέστραπται τὸ λαμπρὸν πρὸς τὸν ἥλιον.

‘(...) and when [the Moon] sets before the sun and when it sets after the sun, the bright (side) is turned towards the sun.’

— Geminus 9.1

Genre: astronomy
Date: 1st century BCE

Geminus. 1975. Géminos, Introduction aux Phénomènes, ed. G. Aujac. Paris: Les Belles Lettres. 58
