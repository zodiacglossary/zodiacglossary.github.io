ŠU₂
rabû

Akkadian verb

Last edited: 6/26/2023, 4:43:13 PM
Primary meaning to set
Literal translation to set
Variants
ŠU₂ rabû
Edited by
Mathieu Ossendrijver
to set at the last appearance synodic phenomenon

30 ME ina IGI-at ŠU₂-šu₂ ša₂ ME 5 ZI-ma ŠU₂

‘30 days before its setting it (Saturn) moves (0);5 per day, then it sets.’

— AO 6477 (BMAPT No. 42) rev. 4

Genre: mathematical astronomy: procedure text
Provenance: Uruk
Date: ca. 250-150 BCE

Ossendrijver 2012 No. 42

to set astronomical event
