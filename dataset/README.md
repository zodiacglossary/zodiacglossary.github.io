# ZODIAC Glossary Dataset (Version 1.0.0)

This dataset contains structured lexical, semantic, and bibliographic information
for the ZODIAC project (Ancient Astral Science in Transformation).

## Contents

- `data.json` — the full glossary dataset exported from the ZODIAC editor
- `metadata.json` — machine-readable dataset description (DCAT + schema.org)
- `VERSION` — dataset version number

## Description

This dataset includes:

- lemma records
- meanings and meaning categories
- spelling variants
- quotations (with provenance and source metadata)
- cross-links
- language definitions
